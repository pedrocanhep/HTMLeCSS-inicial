Trabalhando com listas:

Listas ordenadas (ol)

Com o comando <ol> (ordened lists) podemos criar listas com uma ordem específica, sendo ordenadas em números, letras ou números romanos (o tipo de ordenação pode ser alterado a partir do comando type), sendo 1 para listas numéricas, a / A para listas por ordem alfabética e i / I para números romanos). Além disso, com o parâmetro start podemos delimitar aonde começará a ordem desta lista, sempre indicando o começo a partir de uma posição númerica.

Listas não ordenadas (ul)

Com o comando <ul> (unordened lists) podemos criar listas sem uma ordem específica, tendo estilos como uma bolinha totalmente preta e pintada (<disc>), uma bolinha com uma borda preta e sem preenchimento (<circle>) ou um pequeno quadrado preto totalmente pintado (<square>)(o tipo de ordenação pode ser alterado a partir do comando type).

Listas de Definições

É possível criar listas para definir termos, como em um dicionário, abrindo a tag <dl>, na qual as <dt>'s são os termos e as <dd>'s são as definições.

Exemplos:

HTML
Linguagem de marcação para a criação de conteúdo de um site

CSS
Linguagem de marcação para a criação de design de um site

JAVASCRIPT
Linguagem de programação para a criação de interatividade de um site

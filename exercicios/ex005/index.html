Principais formatações

Negrito/destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica).
Já nesta frase, temos um termo em destaque usando a tag <strong> (semântica)

Visualmente é a mesma coisa, porém o sentido é completamente diferente!

Itálico/ênfase

Nesta frase temos um termo em Itálico usando a tag <i> (não semântica).
Já nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Novamente, visualmente é a mesma coisa, porém o sentido é completamente diferente.

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo com as tegs <big> e <small>

Texto deletado

Podemos adicionar um texto deletado/excluido utilizando a tag <del>

Texto inserido/sublinhado

Nesta frase temos um texto inserido usando a tag <ins> (semântica).
Já nesta frase, temos um texto sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Podemos utilizar textos sobrescritos, principalmente em funções matemáticas como x20 + y8 = 57 usando a tag <sup>

Texto subescrito

Podemos utilizar textos subescritos, como em elementos químicos, ex: H2O usando a tag <sub>
